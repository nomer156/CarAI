# Local AI for CodexCar

## 1. Ollama
- `ollama --version`
- модель: `qwen2.5:7b-instruct`

## 2. Models folder
```powershell
$env:OLLAMA_MODELS='E:\AI\LocalAI\models'
```

## 3. Run local parser server
В корне проекта:

```powershell
npm run ai:server
```

Сервер поднимется на:

`http://127.0.0.1:11535`

## 4. What it does now
- принимает короткую запись владельца
- сначала пытается нормализовать ее в безопасную команду через локальный `Ollama`
- если `Ollama` не ответил, включает локальный rule-based разбор без ИИ
- затем приложение само выполняет разрешенное действие

### Поддерживаемые owner-intent
- `replace_oil`
- `add_part`
- `update_mileage`
- `note_only`

### Примеры
- `поменял масло 5w40`
- `вчера залил 5w40`
- `поставил колодки trw`
- `пробег 128400`

## 5. Important
- без запущенного локального сервера запись все равно сохраняется локально
- AI-парсинг работает только на том ПК, где запущены `Ollama` и `npm run ai:server`
- локальный ИИ не пишет в БД напрямую
- схема всегда такая: `текст -> структура -> валидация -> действие приложения`
