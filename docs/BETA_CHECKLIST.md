# CodexCar Beta Checklist

## Owner
- Войти через Google.
- Завершить регистрацию и сохранить автомобиль.
- Проверить, что экран регистрации больше не появляется при следующем входе.
- Открыть карточку автомобиля, проверить QR и обычный ID владельца.
- Добавить деталь вручную и отредактировать ее.
- Проверить, что деталь осталась после обновления страницы.

## Mechanic
- Войти под аккаунтом механика.
- Ввести ID владельца вручную или считать QR.
- Добавить автомобиль в очередь.
- Перевести запись по статусам: `Подтвердить` -> `В работу` -> `Готово`.
- Добавить деталь от имени СТО.
- Добавить выполненную работу.

## Service Admin
- Войти под аккаунтом админа СТО.
- Добавить сотрудника.
- Добавить владельца по ID в клиентскую базу.
- Проверить, что клиент и очередь появились повторно после обновления страницы.

## Moderator
- Проверить список компаний.
- Проверить список сотрудников по компаниям.
- Проверить логи платформы.

## General
- Проверить светлую и темную тему.
- Проверить работу на телефоне.
- Проверить обновление PWA после нового деплоя.
